This a web application project.
1. Following Technology used for this web application
   Backend:  Based on Java 17, Servlet, JDBC for Postgres v 17.6 Database & Tomcat Server v 9.0
   Frontend: Also used JSP, HTML, CSS and Javascript for view/navigation.

2. Application description as follow: 
    We have used Resource inside context.xml for DataSource connection object.(It will handle multiple users at same time without failure).
    We have one Login page with username & password validation. 
    We also have common menu bar to navigate accross the application. 
    It contains various pages like customer, employee, product & order with its creation, 
    It also displays all the records for each objects in the same page whereas we can edit and delete the same data.
